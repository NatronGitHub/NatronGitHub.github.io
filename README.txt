This directory contains third party files that are necessary to build Natron or the Natron SDK,
but are not available anymore from their original source.
Source distributions can be put in the "src" dir, binaries in the "bin" dir.
GitHub limits the file size to 100Mb, and the repository size to 1Gb.
https://help.github.com/en/articles/what-is-my-disk-quota
