OCIO Support

OpenColorIO is integrated into Natron's native color management system. The program currently runs OCIO version 1.x.

Headless Rendering

Natron projects can be rendered directly from the command line. Render-farm integrations are available for Afanasy, Deadline, and Prism.

Open Source

Natron licensed under the GNU GPL, it is free software and yours to keep forever! Source code is available on GitHub.

Human-Readable Files

All Natron files are saved as plaintext XML.
