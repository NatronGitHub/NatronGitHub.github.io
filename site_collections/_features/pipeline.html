OCIO Support

OpenColorIO (OCIO), the open-source color management solution from Sony Picture Imageworks, is integrated into Natron's native color management system. With OCIO, Natron's profile produces consistent colors across all compatible applications, simplifying the color management process.

Headless Rendering

Natron projects can be rendered directly from the command line and can harness the full power of your render farm, thanks to GPU acceleration. Render-farm integrations are available for Afanasy, Deadline, and Prism.

Open Source

Natron licensed under the GNU GPL license, and developed in the open by a thriving community. It is free software, and yours to keep forever! Source code is available on GitHub.

Human-Readable Files

All Natron files are saved as plaintext XML. File errors are easy to fix - just use any text editor of your choice - and the open file format makes sure that you won't have vendor lock-in.
