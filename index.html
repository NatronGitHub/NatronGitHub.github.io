The tools youʼd expect to be here

The standard 2D compositing toolkit you already know and love.

  • OpenFX Plugin Support

    Natron supports industry-standard OFX plug-ins.

  • 2D Tracking

    Natron includes a fast, pixel-perfect 2D tracker.

  • Roto

    Multi-layered rotoscoping with per-vertex feathering.

  • Keying

    Natron offers various keying and matte refinement tools.

  • Curve & Dope-sheet Editors

    If you've used a curve editor before you know how to use this one too.

  • Headless Rendering

    Natron is capable of running without a GUI can be integrated with render farm managers.

Work smarter with nodes

No more frustrating hunts through nested precomps! Fusion and Nuke users will be right at home in Natron’s node graph. Group, merge, and tinker to achieve just about any effect. Create pixel perfect mattes, cleanplates, and set extensions, all with node trees.

Build, group, and save your own nodes to create custom tools and presets. Natron features drag-and-drop installation of community-made “pyplug” nodes.
